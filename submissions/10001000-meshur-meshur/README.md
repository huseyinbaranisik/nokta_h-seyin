# KodKart Arena 🃏🚀

KodKart Arena, yazılımcıların yeteneklerini dijital kartlar halini getirerek sergilediği, rekabet ettiği ve oyunlaştırılmış bir ortamda (Gamification) gelişimini takip ettiği modern ve eğlenceli bir mobil platformdur.

## 🚀 Teknolojiler ve Mimari
Projemiz tepeden tırnağa modern standartlarda, profesyonel bir altyapı ile geliştirilmiştir:

- **TypeScript:** Tip güvenliği ve ölçeklenebilirlik.
- **Zustand:** Hızlı, modern ve kalıcı (AsyncStorage destekli) state yönetimi.
- **React Navigation:** Kesintisiz ve akıcı ekran arası geçişler.
- **Expo v55:** En güncel React Native geliştirme deneyimi.

## 🎯 Projenin Amacı ve Oyunlaştırma Özellikleri
Projenin temel amacı, yazılım gelişim sürecini bir oyun haline getirerek motivasyonu artırmaktır.

**Oyunlaştırma Mekanikleri:**
- **XP Sistemi:** Yeni beceriler kazandıkça ve projeler tamamladıkça tecrübe puanı (XP) kazanın.
- **Seviye Atlama:** XP biriktirerek "Junior"dan "Senior" seviyesine kadar unvan atlayın.
- **Bilgi Yarışması (Quiz):** "Yetenek Arenası" sekmesinde yer alan Teorik Sınav ve Kod Okuma (Bug Hunter) özellikleriyle bilginizi test edip XP kazanın!
- **Enerji Barı ve Kahve Mekaniği:** Çalıştıkça enerjiniz düşer, kahve içerek enerjinizi doldurabilirsiniz.
- **Liderlik Tablosu (Leaderboard):** Global sıralamada diğer geliştiricilerle rekabet edin.

## ⚙️ Nasıl Çalıştırılır? (Installation & Run)
Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

### 1. Gereksinimler
- Node.js (LTS sürümü önerilir)
- Expo Go (Mobil cihazınızda)

### 2. Kurulum
```bash
# Depoyu klonlayın
git clone https://github.com/MOHAMMEDALMASHHOR/KodKartArena.git
cd KodKartArena

# Bağımlılıkları yükleyin
npm install
```

### 3. Uygulamayı Başlatma
```bash
# Geliştirme sunucusunu çalıştırın
npx expo start -c
```
Terminalde çıkan QR kodu, mobil cihazınızdaki Expo Go uygulaması ile taratarak uygulamayı anında görüntüleyebilirsiniz.
